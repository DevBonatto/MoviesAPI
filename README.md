## ▶️ Running
Clone the project with:
```bash
  git clone https://github.com/DevBonatto/MoviesAPI.git
```

Then use this command to install the dependencies:
```bash
  npm install
```

After installing the dependencies, run this to migrate the database:
```bash
  npm run migrate
```

To start:
```bash
  npm start
```

Develop:
```bash
  npm run dev
```
